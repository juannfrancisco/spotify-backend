
## Description

Proyecto de ejemplo de backend.

Este proyecto utiliza el repositorio Front

```
https://github.com/juannfrancisco/spotify-front
```


```
git clone https://github.com/juannfrancisco/spotify-front
```

```
cd spotify-front
```

```
npm install
```

```
npm run start
```


## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
